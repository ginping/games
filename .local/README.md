# .local

本目录用于本地 agent 开发、调试和临时留存文件，不提交到仓库。

## 目录约定

- `logs/`
  - 通用运行日志
- `dev-logs/`
  - 本地开发调试日志
- `screenshots/`
  - 截图、录屏拆帧、临时视觉验收素材
- `tmp/`
  - 其他一次性临时文件

## 日志建议

- 启动日志优先写入 `.local/logs/` 或 `.local/dev-logs/`
- 按小时分文件，推荐格式：
  - `.local/logs/2026-04-20-14.log`
  - `.local/dev-logs/2026-04-20-14.log`
- 只保留最近 24 小时日志，超出窗口的旧文件应由本地脚本或调试命令定期清理

## 临时资源建议

- 截图、验收图片、录屏导出图优先放到 `.local/screenshots/`
- 不要再把这类临时资源直接放在仓库根目录
- 其他本地调试输出可放到 `.local/tmp/`
